## Deploy

O projeto é um site estático. Para deploy na Vercel:

1. Conecte o repositório Git à Vercel
2. **Framework Preset**: `Other`
3. **Output Directory**: `.` (raiz)
4. Deploy automático a cada push

## Tech Stack

- HTML5 + CSS3 (vanilla, sem frameworks)
- [GSAP](https://greensock.com/gsap/) — animações e parallax
- [Iconify](https://iconify.design/) — ícones (Solar set)
- [Google Fonts](https://fonts.google.com/) — Inter + Playfair Display
